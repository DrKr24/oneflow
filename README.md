# Сайт

_Проект є лендінгом компанії з продажу послуг._


## Технології

  1. HTML5
  2. Tailwind
  3. Vite JS
  4. Google Fonts
  5. gh-pages
  6. vite-plugin-image-optimizer


## Структура
  - style.css | утилітарні класи для tailwind
  - images    | картинки

## Гілки

  1. main     | основні файли проекту
  2. gh-pages | build файли деректорії dist
